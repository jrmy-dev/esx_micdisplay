## Installation

- Download the script
- Move it to your `resources` folder
- Add this in your `server.cfg` :

```
ensure esx_micdisplay
```

- If you want to `change` the microphone image displayed:
  - Go to `img` folder and change `mic.png` to your liking
- Enjoy!
